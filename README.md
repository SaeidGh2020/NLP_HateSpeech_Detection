
# NLP Hate Speech Detection

This project explores hate speech detection using transformer-based models and multiple experimental approaches on the HateXplain dataset.

##  Contents
- Jupyter notebooks with full experiments
- Final research paper
- Academic poster

##  Methods
- BERT / RoBERTa / Transformer models
- Attention analysis
- Multi-approach experimentation
- Dataset: HateXplain

##  Project Structure
See folders: notebooks/, paper/, poster/

##  How to Run
Open any notebook inside `notebooks/` and execute cells sequentially.
